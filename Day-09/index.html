preload: attribute use to allow the developer how the video should be loaded when the page loads, it can have one of three values
such as none, auto, metadata.

none it specify that the browser should not load the video untill the user press play.
auto it specify that the browser should load the entire video when browser load the page.
metadata it specify that the browser should only load the metadata when the page loads such as size,duration,first frame.

src: attribute indicate the path of the video file.

poster: attribute is used to display image

controls: attribute display player controls.

loop: attribute indicate that the the video file should paly repeatedly.
